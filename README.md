# Feature Engineering, Model Optimization & Performance Comparison

## Overview
This project compares multiple Machine Learning regression models using the California Housing Dataset.

## Models Used
- Linear Regression
- Ridge Regression
- Decision Tree Regression

## Techniques Applied
- Feature Scaling using StandardScaler
- Model Training
- Model Evaluation
- Performance Comparison
- Best Model Selection

## Evaluation Metrics
- RMSE (Root Mean Squared Error)
- R² Score

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Google Colab

## Outcome
The models were evaluated and compared based on RMSE and R² Score. The best-performing model was selected after analyzing the results and visualizing the performance comparison.
